# AtliQ Grands – Hospitality Data Analysis (Power BI Project)

## Project Overview
This project focuses on analyzing the business performance of AtliQ Grands, a 5-star hotel chain facing a decline in revenue and market share.  
The goal of this project was to build an interactive Power BI dashboard to generate insights and support data-driven decision-making.

---

## Problem Statement
AtliQ Grands lacked an in-house analytics team and required insights from historical data to:
- Track key business KPIs  
- Identify performance gaps  
- Improve overall decision-making  

---

## Key Learnings

### Business Understanding
- Importance of domain knowledge in data analysis  
- Applied hospitality-specific logic such as defining weekends (Friday & Saturday)  

### Data Modeling
- Structured data using fact and dimension tables  
- Understood how relationships impact performance and scalability  

### Time Intelligence
- Created week-based analysis using WEEKNUM  
- Enabled better tracking of trends over time  

### DAX & Metrics
- Built strong understanding of DAX concepts  
- Used functions like CALCULATE, DIVIDE, WEEKDAY  
- Developed key metrics:
  - Revenue  
  - Occupancy %  
  - ADR (Average Daily Rate)  
  - RevPAR (Revenue per Available Room)  
  - Cancellation %  
  - No-show rate  

### Analytical Thinking
- Focused on converting raw data into actionable insights  
- Understood how analytics supports business decisions
